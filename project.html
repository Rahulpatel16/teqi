PROJECTS

Here you find some of the personal and clients projects that I created

Grocery Order Website

For one of the wholesalers, I built a website. The salesman visits every store and places orders using this website. The website displays the goals that the salespeople must achieve. Additionally, the admin page has CRUD functionality for adding or changing goods.

Form 15CA Generate

Form 15CA is a declaration form available to peoples who require to file for foreign remittances made outside India. As per my client's requirements, i have prepared this software which helps them to convert all Excel data to XML file (i.e Form 15CA Format) as this website only accepts the form submitted in XML Format which has to be uploaded on Income Tax's Website, I have created this software with the help of python.

PDF File Spliting Tool

The client needs an application to separate some PDF files from a folder whose name is present in Excel data. There are around 20 to 30 Customer PDF files to separate from 1000 PDFs in a folder, So I create a python application that takes the name from Excel and separates that PDF from that folder to another folder.

ACK No PDF Download (Automation)

The client requires an software or application that automatically downloads a PDF of ACK from Income Tax Website So I create a python software that gets ACK No from Excel file and put it in the filter box of Income Tax Website and download that PDF, there are around 1000 ACK PDFs to download.

Placement Manangement System

I create a system to manage the placement posted by the company at the college, there are 3 views College, Company, and Student. All the views have their database and login user id and password. A company will post a job and the student can register for that job and college can see how many companies and students are there and it can modify the data also.

Admission form and Duplicate entry remover

This project is given to me in a hackathon, we have to create a web application for student admission, the application checks if the same student registered the form more times so only the latest form will be present and all others will be going to duplicate the database, so more students can register the admission form.

Login CURD System

This is a login page, it will take input and check the userid and password with the database and give access. This application also has functionlity of CURD operations.
